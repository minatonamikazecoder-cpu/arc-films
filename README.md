# ARC Films — Visual Storytelling Studio

A professional website/portfolio showcasing visual storytelling, filmmaking, and production services.

## Tech Stack
- **Structure**: HTML5
- **Styling**: Custom CSS (Vanilla CSS)
- **Interactivity**: Vanilla JavaScript

## Pages Included
- **Home (`index.html`)**: Welcome screen and featured highlights.
- **About (`about.html`)**: Details about the studio and team.
- **Work (`work.html`)**: Portfolio showcasing projects.
- **Services (`services.html`)**: Details of the production services offered.
- **Contact (`contact.html`)**: Contact details and inquiry form.
